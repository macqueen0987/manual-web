## 필독 (URL·게시 동작)

| 구분 | 설명 |
|------|------|
| **버전 URL slug** | 게시 시 입력하는 **버전 이름**(예: `2026.06.03`)과 URL slug는 **다름**. slug는 `slugifyVersionName()`으로 만든 **8자 해시**(예: `f4a15ed7`). 공개 URL: `/{productSlug}/{해시slug}` |
| **게시 모달** | Admin → 「게시하기」→ 버전 이름만 입력. **slug 미리보기 UI 없음** — 게시 후 Admin 버전 행의 「보기」 링크로 실제 URL 확인 |
| **문서 slug** | 비라틴 제목(예: `시작하기`) → `page-{해시8}` (예: `page-8779a905`) |
| **에디터 URL** | `.../latest/editor` = 작업 중. 게시본 선택 시 `.../{versionSlug}/editor` (해시 slug) |
| **시드 제품** | Blue/beSRT 등은 `latest`만 있을 수 있음 → **게시 전** 공개 문서 없음. 점검 시 한 제품은 반드시 **게시**할 것 |
